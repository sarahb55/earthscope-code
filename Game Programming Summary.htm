Game Programming Summary

 

Total Time 3-4 Days 50-80% ~12-16 hours

 

Summary

First step was to design and create a preliminary volcano with no graphics. I created this using ActionScript 3.0.

 

Design – Created preliminary design and implemented in ActionScript 3.0. Design consists of three types of classes based on the model-view-controller design pattern. These are GameEngine, GameData, and GameView.

 

The GameEngine is the link between the screen and the back-end of the game. The GameEngine has a Timer that is set to regular intervals. Each time it goes off it updates the GameData objects by calling a method each game piece has called tick.

 

After each piece has been updated the appearance the game is updated in the View classes if updateView from the data object returns true. There can be more than one view for a given set of data.

 

Adding the Graphics:

After I had a few tiff files to work with – I used flash to create a MovieClip of a puff of smoke. Then I added the movie clip to the game. I also added the start and pause buttons and a debug window. (This took about 2 hours.) Once this was complete, I published it on the web to test
